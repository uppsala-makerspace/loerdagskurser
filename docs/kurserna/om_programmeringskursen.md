# 🇸🇪 Om programmeringskursen 🇬🇧 About the programming course

=== "🇸🇪"

    Programmeringskursen är en av den [kurser](README.md)
    av [Lördagskurserna](https://uppsala-makerspace.github.io/loerdagskurser/).

    Under programmeringskursen kursen lär man sig att programmera.

=== "🇬🇧"

    The programming course is one of [the courses](README.md)
    of [the Saturday courses](https://uppsala-makerspace.github.io/loerdagskurser/).

    In the programming course one learns how to program.

![Processing med kod](processing_med_kod.png)

=== "🇸🇪"

    > Programmeringskod i vårt första programmeringsspråk kallad 'Processing'

=== "🇬🇧"

    > Programming code in our first programming language called 'Processing'


=== "🇸🇪"

    Att programmera är att berätta till en dator vad den måste göra,
    t.ex. att låta en användare spelar en datorspel.

    Senare i kursen kommer ni att göra [`git` kursen](om_gitkursen.md)]
    för att kunna arbeta tillsammans med ett programmeringsprojekt.

    Kursen använder kursmaterialet
    [Processing för ungdomar](https://github.com/richelbilderbeek/processing_foer_ungdomar)

=== "🇬🇧"

    Programming is telling a computer what to do,
    for example, letting a user play a computer game.

    Later in the course, you will do [the git course](om_gitkursen.md)
    to be able to work on a programming project together.

    The course uses the (Swedish-only!) course material
    [Processing för ungdomar](https://github.com/richelbilderbeek/processing_foer_ungdomar)
