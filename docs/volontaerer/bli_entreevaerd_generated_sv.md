
# Hur bli man entrévärd hos Lördagskurserna?

[![Lördagskurserna logo](loerdagskurser_logo_64_x_60.png)](https://uppsala-makerspace.github.io/loerdagskurser/volontaerer/bli_entreevaerd/)

## Hur kan jag blir entrévärd?


Med att säger detta till oss :-)

Bäst är att [kontakta kurs samordnaren](https://uppsala-makerspace.github.io/loerdagskurser/kontakta_oss)
så att det finns gott om tid för all dina frågor och idéer.

Annars är det att dyka upp under Lördagskurserna
och, när våra gäster har tagits hand om,
snacka med den dåvarande värd.
